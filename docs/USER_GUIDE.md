 # PipelinePro User Guide (Quick)
 
 This guide explains the main workflows for everyday use.
 
 ## 1) Sign In
 - Open the application URL provided by your admin.
 - Enter your username or email and password.
 - If you were given a temporary password, change it immediately after login.
 
 ## 2) Dashboard Overview
 - Review high-level KPIs and recent activity.
 - Use quick links to jump to active accounts, opportunities, or proposals.
 
 ## 3) Accounts & Contacts
 - Create or open an account.
 - Add contacts to the account (email, phone, role).
 - Use the account detail page to view linked opportunities.
 
 ## 4) Opportunities (Pipeline)
 - Create an opportunity from the Opportunities page.
 - Set value, owner, stage, and PWin.
 - Link the opportunity to an account.
 - Track updates in the activity timeline.
 
 ## 5) Teaming Partners
 - Create a partner record in the Teaming page.
 - Enter capabilities and contract vehicles.
 - Review fit score and NAICS alignment.
 
 ## 6) Market Intel (SAM.gov)
 - Search by keywords, NAICS, or agency.
 - Review results and open relevant opportunities.
 - Use results to seed new opportunities.
 
 ## 7) Proposals
 - Create a proposal from an opportunity.
 - Configure proposal volumes (exec summary, technical, management, past performance).
 - Upload documents and supporting files as needed.
 
 ## 8) Proposal Workspace
 - Open the Proposal Workspace for guided steps.
 - Generate or refine sections using the AI assistant.
 - Export final artifacts when ready.
 
 ## 9) PTW (Price-to-Win)
 - Create scenarios for labor and price modeling.
 - Compare scenarios side by side.
 - Use outputs to support pricing decisions.
 
 ## 10) PWin
 - Track PWin trends by opportunity.
 - Use the data to prioritize pursuit decisions.
 
 ## 11) Compliance Reports
 - Generate FedRAMP, NIST 800-53, or CMMC reports.
 - Export compliance summaries as JSON or CSV.
 
 ## 12) Admin Tasks (Admins Only)
 - Manage users and roles.
 - Reset user passwords (one-time temp password shown on reset).
 - Manage AI providers and tenant settings.
 - Review audit logs for governance.
 
 ## Tips & Best Practices
 - Keep opportunities linked to accounts for full context.
 - Use consistent naming for proposal volumes.
 - Reset temporary passwords immediately.
 - Export compliance reports before reviews or audits.
